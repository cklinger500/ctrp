CTRP Style Guide

This is the CTRP application styleguide using Fett, a lightweight pattern renderer in Javascript. The CTRP application front-end was implemented using the Bootstrap framework and AngularJS 1.x. This guide does not provide information about the default Bootstrap framework or core AngularJS functionality. Rather, the guide will serve as a reference for site specific styles and custom CSS and JS/AngularJS implementations.

Components

Swatches

Colors can be shown using swatches, which show the color itself, a name, and the code to generate it.

Code Blocks

Example code can be given in code blocks for HTML or JS. These blocks are automatically syntax highlighted depending on the language defined with the language-markup or language-javascript classes.

					
						

							
								
This is an example of an HTML code example!

							
						

					
				
					
						// Different languages are highlighted differently
						document.write('and here is javascript');

Hidden Code

Sometimes you want to display an example, but you don't want to display code, or you want to display different code. To do this, just add the hidden class to an example.

						
							
Click here.

							
This HTML is shown in the example, but the source isn't rendered below. This can be useful when code requires some shimming to work properly in the context of the styleguide, or when some of the setup required to run code is irrelevant to the documentation.

						
					
						
	var times = 0;
	document
		.getElementById('click_me')
		.addEventListener('click', function(){
			this.innerHTML = ['Clicked', ++times, 'times'].join(' ');
		});
